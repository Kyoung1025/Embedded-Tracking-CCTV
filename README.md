# 임베디드 최종 과제 - Embedded-Tracking-CCTV

## 실행방법

### 1. 깃 저장소 클론(다운로드)
```Shell
git clone https://github.com/Kyoung1025/Embedded-Tracking-CCTV.git
```

### 2. 다운로드 폴더로 이동
```Shell
cd Embedded-Tracking-CCTV
```

### 3. 가상 환경 생성 및 활성화
```Shell
python -m venv .venv
```
```Shell
.\.venv\Scripts\activate
```

### 4. 라이브러리 설치
```Shell
$ pip install -r requirements.txt
```

### 5. 서버 실행
```Shell
python server.py
```
